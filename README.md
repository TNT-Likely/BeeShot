# BeeShot

免费开源的 App Store 截图生成器，轻松制作带设备框的宣传图。

## 特性

- 支持多种设备尺寸 (iPhone 6.7", 6.5", 5.5" 等)
- 自定义标题和副标题
- 丰富的渐变背景预设
- 批量导出为 ZIP
- 无水印、无需注册
- 纯前端实现，数据不上传

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 技术栈

- React 19
- TypeScript
- Vite
- html2canvas
- JSZip
